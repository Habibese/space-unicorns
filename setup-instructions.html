You need to add your Stripe API keys to use the payment system.

1. Get Stripe Keys

Go to Stripe Dashboard and get your:

  • Publishable Key (starts with pk_test_...)
  • Secret Key (starts with sk_test_...)

2. Edit .env File

Open the .env file and replace the placeholder keys:

STRIPE_PUBLISHABLE_KEY=pk_test_YOUR_KEY_HERE
STRIPE_SECRET_KEY=sk_test_YOUR_KEY_HERE

3. Restart Server

Stop the server (Ctrl+C) and restart it:

npm start
